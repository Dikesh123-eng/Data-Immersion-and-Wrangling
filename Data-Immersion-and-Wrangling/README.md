# 📊 Data Immersion & Wrangling Project

## 🔹 Objective
To clean and prepare raw sales data for analysis.

---

## 🔹 Steps Performed

### 1️⃣ Data Quality Assessment
- Checked missing values
- Identified duplicate rows
- Detected inconsistent date formats
- Found outliers in Quantity column

### 2️⃣ Data Cleaning
- Removed duplicate records
- Filled missing values using mean
- Standardized date formats
- Removed extreme outliers

### 3️⃣ Feature Engineering
- Created Total_Sales column
- Created Customer_Age column

---

## 🔹 Tools Used
- Python
- Pandas

---

## 🔹 Output
A clean dataset named `cleaned_sales_data.csv` ready for analysis.
